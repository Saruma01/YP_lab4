Цель: изучить работу алгоритмов поиска подстроки в строке
Задачи:
- освоить работу со строками в языке C++
- изучить алгоритмы поиска подстроки в строке
Задание:
1. Скачать книги про Гарри Поттера в формате TXT, например, отсюда
https://github.com/bobdeng/owlreader/tree/master/ERead/assets/books
Чтобы это сделать быстро, можно скачать весь репозиторий в формате ZIP отсюда
https://github.com/bobdeng/owlreader/tree/master
и затем найти все книги в каталоге ERead/assets/books
2. Реализовать следующие алгоритмы поиска подстроки в строке
	- Наивный поиск: перебираем позицию начала возможного вхождения образца в текст, после чего проверяем совпадает ли нужный кусочек текста с образцом.
	- Через метод find у std::string
	- Через функцию strstr
	- Через префикс-функцию (алгоритм Кнута-Морриса-Пратта)
	- Через полиномиальное хэширование (алгоритм Рабина-Карпа)
Данные алгоритмы нужно адаптировать так, чтобы находились все вхождения образца в текст.
3. Для одной из книг про Гарри Поттера (желательно брать размером хотя бы 900КБ, текст книги загружается в программу чтением из файла) всеми реализованными алгоритмами найти количество вхождений следующих подстрок:
Harry
I dunno
What's up?
Hermione and Ron
	said Professor McGonagall
и вывести на экран. Если алгоритмы реализованы корректно, они все выдадут одинаковый ответ.
4. Сделать замер времени работы для каждого алгоритма и также вывести на экран. Сравнить эффективность алгоритмов (словесно и в отчете).
